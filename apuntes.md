## Codigos:
### En CMD

- Crea una directorio con "mkdir (Nombre)"
- "CD" al directorio para poder conectarlo a git
- Usando "clone" y el enlace al directorio creado en git
- escribe "code .", a continuacion se abrira VSC

### EN VSC
- crea un archivo .LEEME
#### En la consola de comandos de VSC:
- git version
- git init: Inicializar git
- git status -s: para saber que archivos no tienen seguimiento 
- git add .: para hacer seguimiento del archivo del repositorio
- git commit -m "primer commit"
- git push: para subirlo a la nube
